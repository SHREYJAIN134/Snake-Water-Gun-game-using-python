# Snake-Water-Gun-game-using-python
# Snake Water Gun - Game using Python We all have played Snake,Water &amp; Gun in our childhood.   This Python program brings that nostalgic game back to life.

Game Rules:
-Snake drinks Water  → Snake wins!
-Water damages Gun  → Water wins!
-Gun kills Snake → Gun wins!
-If both players choose the same → It’s a Draw!

How the Program Works:
-The user inputs their choice: snake,water, or gun
-The computer randomly picks one of the three
-The result (Win/Lose/Draw) is displayed instantly
-At the end of each round, you can choose to continue or quit
-Handles wrong spellings and invalid inputs gracefully
